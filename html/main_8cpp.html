Parameters:

Returns:

The purpose of this program is to provide preliminary structural information on biomolecules optimized using Gaussian computational chemistry software. Structural and chemical properties identified include: element identification, bond length, bond order, central angles, and torsional angles. Input of the file to be analyzed will result in an output file listing all the structural information of the biomolecule.
